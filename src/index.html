Welcome to the Clangd Auto-Parser and Format File Generator, or CAPFFG.
Simple mode will walk you through creating a new .clang-format file in a simplified way, combining similar settings into a step-by-step process suited for anyone new to Clangd formatting.
Advanced mode will give you an up-front list of all settings.
Parse allows you to paste in code to automatically divine the set of rules to apply. You can still alter all the same rules you normally can. Any rules that cannot be inferred will be flagged for manual review.
